## 🚀 Tecnologías Utilizadas

### Frontend
| Tecnología | Versión | Descripción |
|------------|---------|-------------|
| Next.js | 16.2.4 | Framework React con App Router |
| React | 19.2.4 | Biblioteca UI |
| TypeScript | 5.x | Tipado estático |
| Tailwind CSS | 4.x | Estilos utilitarios |
| shadcn/ui | - | Componentes accesibles |
| TanStack Query | 5.100.7 | Manejo de estado y caché |
| Framer Motion | 12.38.0 | Animaciones |
| Recharts | 3.8.1 | Gráficos y visualización de datos |
| Lucide React | 1.14.0 | Iconos |

| PostgreSQL | 15.x | Base de datos relacional |
| JWT | - | Autenticación |
| Bcrypt | - | Encriptación de contraseñas |
| Puppeteer | - | Generación de reportes PDF |
